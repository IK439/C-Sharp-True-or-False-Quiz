# True or False Quiz (C#)

## Overview
This is a simple **True/False quiz** program written in **C#**. It asks the user a series of questions, validates their input, stores their answers, and then calculates and displays their score. The program is designed for console-based interaction, where the user answers each question with either "true" or "false".

## Features
- Displays a set of True/False questions.
- Validates user input to ensure only "true" or "false" responses.
- Calculates the user's score based on their answers.
- Displays the user's score at the end of the quiz.

1. Make sure you have [.NET SDK](https://dotnet.microsoft.com/download) installed.
2. Create a new console project or use this code in an existing one.
